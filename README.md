# eda-employee-dataset
Complete Data Science workflow project using Pandas and Matplotlib.   Covers data loading, cleaning, feature engineering, filtering, sorting, aggregation, visualization, and business analytics. Includes a mini analysis report and a bonus weighted ranking challenge.



📌 Project Title
Employee Data Analysis (Pandas + Matplotlib)

🎯 Objective
Build a complete Data Analysis workflow used by Data Scientists and ML Engineers — from raw CSV data to insights and visualizations.

📂 Dataset
employees_data.csv (synthetic dataset of 1000 employees)

🛠 Tech Stack
Pandas → Data cleaning, feature engineering, aggregation

Matplotlib → Visualizations)

📑 Workflow
Pandas Basics → load, inspect, summarize data

Data Cleaning → missing values, duplicates, type conversion

Feature Engineering → TotalIncome, ExperienceLevel, PerformanceGrade

Filtering & Sorting → queries on city, department, salary, performance

GroupBy Analytics → department, city, gender, education insights

Aggregations → highest salary, correlation matrix, std dev

Advanced Pandas → value_counts, query, merge, pivot_table, crosstab

Matplotlib Visualizations → histograms, scatter plots, line plots, bar charts
